# Axionix X

A modern, high-performance company website for **Axionix X** — an innovative tech and skill development company providing digital services and professional training programs.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (animations)
- **next-themes** (dark/light mode)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

- `src/app/` — App Router pages and layout
- `src/components/layout/` — Navbar, Footer
- `src/components/sections/` — Hero, Services, Programs, Why Choose, News, CTA, Contact
- `src/components/ui/` — Reusable UI (e.g. WaveDivider)
- `src/components/providers/` — ThemeProvider

## Features

- Sticky navbar with active section highlight and animated mobile menu
- Hero with gradient background and floating shapes
- Services grid with hover lift and glow
- Skill development programs with animated counter
- Why Choose, News & Events, CTA, and Contact sections
- Dark/light mode toggle
- Smooth scroll and Framer Motion scroll-triggered animations
- Responsive, accessible, SEO-friendly

## Build

```bash
npm run build
npm start
```

## Brand

Primary accent: `#58c9ff` (Axion cyan). Used for CTAs, borders, and glow effects.
